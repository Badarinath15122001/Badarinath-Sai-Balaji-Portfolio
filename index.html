Projects

Stack Overflow Label Prediction Using Machine Learning

Built a machine learning model to predict relevant tags for Stack Overflow questions using logistic regression with a OneVsRest classifier. This improved searchability and response time, enhancing the user experience and increasing the accuracy of question classification.

Technologies Used

Python Scikit-learn NLTK Pandas NumPy SQLite Jupyter Notebook
GitHub Repository

Problem Statement

Automate tag prediction for Stack Overflow questions to improve searchability and response accuracy, making it easier for users to find relevant content.

Solution

  • Developed a multi-label model using OneVsRest Classifier and Logistic Regression.
  • Preprocessed text using NLTK and Snowball Stemmer to clean and standardize data.
  • Used TfIdf Vectorizer to convert text into numerical format, enhancing model accuracy.
  • Evaluated performance using accuracy, precision, recall, and F1-score metrics.
